What was the problem?

What problem solving techniques did you use?

As I've never done this before I decided to check other testers reports. Unfortunately there was no information about how to actually get the logs. How did you feel throughout the process?

I felt stuck at first, but the closer I was to find a solution the more excited I was.

What did you learn?

I learned that I can solve the problem on my own if I need to.
